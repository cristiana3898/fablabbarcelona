    • Bio Plastic fabric

  • To create a Bio plastic fabric to produce an apron you have to take the recipe and the ingredients and some containers.
  • The ingredients:
  • -250ml of water,
  • -50g of gelatine,
  • -20g of glicerina vegetal.
  • -Acrylic paint
    • We put the water in the pot on the electric cooker, then we started add the gelatine while mixing it, we have to be careful to don't let form lumps in the liquid. We let it boil and then we put the glicerina. Then the gelatine started create a white foam that we remove with a spoon. Then we add the acrylic paint and mix it all over gain. After taking away the foam we inverted the liquid on a table and let it dry for some days.

    Ingredients and Tools

    Acrylic paint, Water, Glicerina Vegetal, Gelatine, Balance, Containers, Pot and Electric Cooker.

    • 3D Printer

  • 3D Printing, also known as Additive Manufacturing, refers to processes used to create a three-dimensional object in which layers of material are formed under computer control to create an object. Objects can be of almost any shape or geometry and typically are produced using digital model data from a 3D model or another electronic data source such as an Additive Manufacturing File (AMF) file. STereoLithography (STL) is one of the most common file types that 3D printers can read. Once you have to file, you must open it with Cura.     Arduino

  • Arduino is an open source computer hardware. It is composed by electronics boards that it has a microcontroller. The boards are equipped with sets of digital and analog input/output (I/O) pins that may be interfaced to various expansion boards (shields) and other circuits. The boards feature serial communications interfaces, including Universal Serial Bus (USB) on some models, which are also used for loading programs from personal computers.

    • Milling Machine

  • Milling is the machining process of using rotary cutters to remove material from a workpiece by advancing in a direction at an angle with the axis of the tool. With this technology you can create your own circuit. First step you have to open your PNG file on Fab Modules that is an open Source CAM software. It allows you to generate machining strategies for lots of different processes and machines. Once you have finished setting up the cutting strategy you should save th file (pay attention to the z0 and the zHome, it have to be at leat 5 or you'll cut when it returns to the home). It'll download into your computer as a .rml file. Then you have to open the file with Roland Monofab that is a desktop milling machine. It moves in much the same way as a 3d printer, but instead of adding material it substracts it. The CAM software that controls the machine is called V panel. First of all we'll need to secure the Blnck PCB to the bed of the monofab with double sided tape. Then by moving the Milling bit at the bottom left corner of the blank we can Zero the X and Y axis using the User oordinate system.
