How to do a screw (thread and nut) on Onshape

    First of all go to Onshape, sign in and go in the document menu, then in workspace units and select millimiters.
    To create the thread:

    • create sketch 1 -> select top plane -> create a circle and click on done
    • click extrude (solid, face of sketch 1)
    • Helix (turns, face of extrude 1)
    • create a plane (if it is necessary) -> select plane, offset
    • create sketch 2(hide part 1) -> front plane-> create a triangle (make sure that the distance from the angle and the helix is 0) -> select =(equal) and click all the faces -> select a face in up and the helix and click on pierce
    • click on sweep -> solid, add, face of sketch 2 merge with all

      • Duplicate and make the tringle bigger
      To create the nut:
    • click on derived -> the copy bigger
