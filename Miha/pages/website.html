How to create a website

    For the first step we have to create an account on Trello that is a project management tool where you can organize work projects, family chores, travel plans, or just about anything else. For exemple you can organized it in things that are:

    • baked
    • to do
    • doing
    • done

    also we have to create an GitHub account and download GitHub Desktop, it is a control repository. It is mostly used for code an it offers both plans for private and free repositories on the same account which are commonly used to host open-source software projects. With this application we can save and modify files whenever we want.

    So now we have to create a repository (pushing the plus incon near at the imagin of your account). After we have to set up it in desktop, open, clone and add it at GitHub.

    Finally we have to edit our web page with Sublime, it is a text editor for code, markup and prose.It natively supports many programming languages and markup languages, and functions can be added by users with plugins, typically community-built and maintained under free-software licenses.
