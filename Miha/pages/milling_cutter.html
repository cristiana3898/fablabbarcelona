The milling cutter

Milling cutters are cutting tools typically used in milling machines or machining centres to perform milling operations (and occasionally in other machine tools). They remove material by their movement within the machine or directly from the cutter's shape. It can be used also to create circuits like we do.

To create files/circuits for the milling the software called EAGLE PCB Design or also downloadig png files in b&w or trasparent (the machine will remove the black).

  • Open the web page www.fabmodules.org and click on Input Format;
  • Choose Image (.png) and Upload Image;
  • Click on Output Format and choose the machine (Roland Mill (.rml));
  • Setting machine: machine SRM-20; Xmin 0; Ymin 0;
  • Click on Process and choose (PCB traces(1/64)) or (PCB outline(1/32));
  • Change Zhome to a personal setting;
  • Click on Calculate, and after Save (it will be saved in .rml).

  • Now we have to set up the machine, putting a double face scotch tape on the material and paste it on the superficial layer. After select the cutting tip (-32 for outline/-64 fot the traces)
  • Set up the origin where is needed
  • Select cut and add the file
