The 3D printer

    3D Printing is a processes used to create three-dimensional objects, where layers by layers of material formed an object under the computer control.
    Objects can be downdloaded or created by a 3d softwere and have to be saved in .STL (STereoLithography) or .3dm. Once you have the file open it with Cura, or another similar softwere, where you have to select the machine that you'll use and the material. Then check all the values, in particular you have to controll that

    • the layer's height are in crescents values;
    • if the wall thickness is what you want
    • the infill density is in the right percentual
    • the temperature (generaly for PVL the printing temperature is 210/250ยบ and the buil plate temperature is almost 45ยบ
    • check if the object need support
    • create a brim (initial layers at the bottom to create a tipe of support for all the object)

    and then save it. (For some machine, especially in the Multimakers, you have to put the file in the memory card).
    After you have to prepare the machine:

    • go to prepare -> setup -> preheat (in this fase you put out some material to be sure that is melted)
    • prepare -> Auto home or set the z (if the z is not at the right high you can go to Prepare -> disable steppers and regulate manually the z asset
    • after you have to put the hair spray on the surface of the glass
    • then you have to stuck it on the base with some pins
    • now select the file on the machine and wait until it's done.
