How to create a Bioplastic textile

The Bioplastic is an articial fabric made by natural materials like:

  • Glycerin (20 g)
  • Gelatin (50 g)
  • Water (250 ml)
  • almost any kind of painting (as required)
(the recipe can be multiplied as many times we need)

This textile can be creat at home with tools like:

  • Bilance
  • Burner
  • Pot
  • Containers
It is very simple to do.

First of all we have to put the water in the pot and turn on the burner, immediatly after we have to pour the gelatin in the water very slowly to prevent clumps and blend it some minutes, then we can add the glycerin.

After this step we can see apper, in the upper part, a kind of white glaze that can be removed or keeped (if you want to have a pattern you can keep it because this foam can change and create a shade of the colour), then we can add colours if we want.

When the recipe is done we have only to put it on a plain surface and let it chill (remeber: the bioplastic is liquid so it need to have a frame to stop its expantion)
