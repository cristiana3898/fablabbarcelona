Bioplastic Textile

In this second day we helped a member of the FabLab to create an apron in bioplastic textile. First of all we took a base and we created the wood's edges to not drop the liquid. Then we prepared the ingredients and the tools that we used.

Ingredients:

  • Acrylic paint;
  • Glycerin vegetal;
  • Gelatine;
  • Water.
    Tools:
  • Balance;
  • Borner;
  • Pot
  • Containers.

We put the pot on the borner and then we mixed water and gelatine. If you want a clean work, you have to pay attention that the liquid doesn't have lumps. Sometimes, after a few minutes, it can form a white resin that it have to be removed if you don't want a pattern. After we mixed the liquid with the acrylic paint and then we overthrown the liquid on the base that we created.

Laser Cutter, Trotec

Laser cutter is a technology that permits to cut materials, and it is used for industrial manufacturing application, and also for school, business and hobbyists. Today a member of the FabLab explained to us how to use the laser cutter. First of all you need to check that the 2 tube are connected with the machine. Then you have to control that the removable table is hooked well. At this point you can switch on the machine. After you have to find the exaclty point of the table, for doing this you can move to right and left the table through the keys and also to up and down. For the lasest passage, you can use the focus tool. The focus tool serves to understand to machine the thickness of the material that you use. Then, through the computer, you have to decide the working area. At the end you have to send the file.

Process:

  • Check the aspirations tube and the cable;
  • Turn on the machine;
  • Focusing using the tool;
  • Download the 3dm file from the cloud;
  • Open the file with Rhino;
  • On Rhino, check if the design have break lines, if it has, you hAve to select all the design and join it (clicking on join);
  • On Rhino, check the measures and put all the design in red;
  • Click on file and then on print;
  • Check id the machine is the right one, then check the scale that have to be 1:1, then check if the imagine is in vectors:
  • Click on properties and select the working area;
  • Then click on window, when it opens, on the top click move and move the area on 0.0;
  • Click on print;
  • Open Tropec;
  • Drag the design from the left to the middle;
  • Click on ready to connect the machine to the pc;
  • Click on setting and then on materials (first voice);
  • Select your material, speed and power (you have to take some tests to find the right one.

Arduino

Arduino is an open source computer hardware. It is composed by electronics boards that it has a microcontroller. The boards are equipped with sets of digital and analog input/output (I/O) pins that may be interfaced to various expansion boards (shields) and other circuits. The boards feature serial communications interfaces, including Universal Serial Bus (USB) on some models, which are also used for loading programs from personal computers. The Arduino project started in 2003 as a program for students at the Interaction Design Institute Ivrea in Ivrea, Italy, aiming to provide a low-cost and easy way for novices and professionals to create devices that interact with their environment using sensors and actuators. Common examples of such devices intended for beginner hobbyists include simple robots, thermostats, and motion detectors.

3D printer

3D Printing, also known as Additive Manufacturing, refers to processes used to create a three-dimensional object in which layers of material are formed under computer control to create an object. Objects can be of almost any shape or geometry and typically are produced using digital model data from a 3D model or another electronic data source such as an Additive Manufacturing File (AMF) file. STereoLithography (STL) is one of the most common file types that 3D printers can read. Once you have to file, you must open it with Cura. On it you have to select the material and the machine that you'll use. Then you have to check all the values and save it. (For the Multimaker, you have to put the file in the memory card). To prepare the machine you have to put the hair spray on the surface of the glass top, the you have to stuck it on the base with some pins. Then select print on the machine and it'll pepare the material. Then wait until it's done! Below there're the values of our object.

Milling Machine

Thanks by this machine you can create a cicuite. Milling is the machining process of using rotary cutters to remove material from a workpiece by advancing in a direction at an angle with the axis of the tool. The milling cutter is a rotary cutting tool, often with multiple cutting points. As opposed to drilling, where the tool is advanced along its rotation axis, the cutter in milling is usually moved perpendicular to its axis so that cutting occurs on the circumference of the cutter. First step you have to open your PNG file on Fab Modules that is an open Source CAM software. It allows you to generate machining strategies for lots of different processes and machines. Once you have finished setting up the cutting strategy you should save th file (pay attention to the z0 and the zHome, it have to be at leat 5 or you'll cut when it returns to the home). It'll download into your computer as a .rml file. Then you have to open the file with Roland Monofab that is a desktop milling machine. It moves in much the same way as a 3d printer, but instead of adding material it substracts it. The CAM software that controls the machine is called V panel. First of all we'll need to secure the Blnck PCB to the bed of the monofab with double sided tape. Then by moving the Milling bit at the bottom left corner of the blank we can Zero the X and Y axis using the User oordinate system.
